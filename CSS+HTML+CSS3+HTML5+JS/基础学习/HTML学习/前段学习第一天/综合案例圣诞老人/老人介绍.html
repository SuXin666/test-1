2．礼物为什么要装在袜子里？

圣诞老人并非要把礼物放在袜子里，来源于一个古老的传说。传说有个心底善良的没落贵族，生活非常艰难。家有三个女儿，因为快要出嫁了但是买不起嫁妆而难过，圣尼古拉斯（圣诞老人的原型）想帮助他们，就从他家烟筒里丢下了三袋金子。金子掉进了挂在壁炉边的袜子里。于是，圣诞节的礼物放在袜子中就这样流传了下来。

3．“圣诞节快乐”英文为何是Merry Christmas而不是Happy Christmas?

据说Merry Christmas 最早出现在1565年，来自一名海军元帅写在信里的祝福。但是Merry Christmas在英语圈流行，是由于英国大文豪狄更斯的大力推广。1843年，狄更斯的中篇小说《圣诞颂歌》发行后，变立刻成为了畅销书，书中就有Merry Christmas一词，当时的商人顺势推出印有Merry Christmas 的圣诞卡，引发抢购潮。因此，Merry Christmas 的说法便随着圣诞卡传遍全世界。 有趣的是，英国女王伊丽莎白从来不说Merry Christmas，只说Happy Christmas,因为Merry 直译为天性活泼，有些微醉的意思，在维多利亚时期，这个词主要混迹在社会底层，意思是以酗酒为乐。Merry Christmas 如果按照字面意思理解的话，就是“圣诞喝到嗨”！对于高贵的女王来说，Happy Christmas更符合女王的身份。

4.圣诞老人为什么架着驯鹿而不是雪橇犬？

据说圣诞老人生活在北极，那里有一种特殊的鹿叫驯鹿。驯鹿适应雪地生活，他们有夸大的蹄子，可以避免陷入雪中，还能拨开雪堆找食物吃，他们的眼睛可以变色，避免雪地反射的强光刺伤眼睛。他们性格温顺，易于驯养，能够轻松穿越马匹难以穿越的雪地，沼泽和森林，因为，圣诞老人选择了驯鹿拉雪橇。

5．圣诞树顶上挂着的星星是什么星星？

圣诞树顶上的那颗星星叫伯利恒之星，是耶稣出生时头顶上冒出来的超新星。

8.《铃儿响叮当》本来是一首感恩节的歌曲

《铃儿响叮当》最初并不是为了圣诞节所写，而是为了感恩节所作。1857年，词曲作家詹姆斯·罗德·皮尔彭特的父亲在一所学校任职，这首歌是写给父亲的学生们，帮助他们参加感恩节的演出。难怪我们在歌词中找不到耶稣的影子，也许正是因为歌词和圣诞节的场景很相似，而且曲子也很欢快，使得这首歌曲成为了150年来最重要的圣诞节保留曲目，也是世界知名度最高的歌曲。
